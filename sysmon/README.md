# Plasma 6 Sysmon Widget

A beautiful, transparent system monitor widget for KDE Plasma 6.

## Installation

Run the `install.sh` script to install the Plasmoid and set up the Python backend to run automatically.

```bash
./install.sh
```

## Manual Installation
1. Install the plasmoid:
   `kpackagetool6 -t Plasma/Applet -i plasmoid`
2. Run the backend:
   `./launch_sysmon_backend.sh`
